# TR.BIDSsvMOD.dgoCtrlUSB
Communication bridge between BVE and Controller (on the BIDS Platform)

## How to Use
BIDS_Serverでmodとして読み込ませてください。事前にドライバのインストールが必要です。

## Using Library
[LibUsbDotNet](https://sourceforge.net/projects/libusbdotnet/)
